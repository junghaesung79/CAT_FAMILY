# Cat Family 🐱

다양한 고양이 품종에 대한 상세 정보를 탐색하고 발견할 수 있는 현대적인 웹 애플리케이션입니다. 검색, 필터링, 정렬, 즐겨찾기 관리 등의 기능을 포함합니다.

## ✨ 주요 기능

- **품종 탐색**: 상세 정보가 포함된 광범위한 고양이 품종 컬렉션 탐색
- **검색 및 필터링**: 크기, 털 유형, 수명, 활동 수준 등으로 품종 필터링
- **정렬**: 다양한 기준으로 품종 정렬 (이름, 크기, 수명 등)
- **즐겨찾기**: 빠른 접근을 위해 좋아하는 품종 저장
- **상세 정보**: 각 품종에는 다음이 포함됩니다:
  - 기원 및 역사
  - 외모 및 신체적 특성
  - 성격 및 기질
  - 건강 정보 및 유전적 위험
  - 케어 및 그루밍 가이드
  - 아이 및 다른 반려동물과의 호환성
- **반응형 디자인**: 모바일 및 데스크톱 기기에 최적화된 인터페이스
- **무한 스크롤**: 더 나은 사용자 경험을 위한 점진적 품종 로딩

## 🛠️ 기술 스택

- **React 18** - UI 라이브러리
- **TypeScript** - 정적 타입 지정
- **Vite** - 빌드 도구
- **React Router** - 라우팅
- **Tailwind CSS** - 유틸리티 스타일
- **Context API** - 상태 관리

## 📦 설치

1. 저장소 클론:
```bash
git clone <repository-url>
cd CAT_FAMILY
```

2. 의존성 설치:
```bash
npm install
```

## 🚀 사용 방법

### 개발

개발 서버 시작:

```bash
npm run dev
```

애플리케이션은 `http://localhost:5173`에서 사용할 수 있습니다.

### 빌드

프로덕션 빌드 생성:

```bash
npm run build
```

컴파일된 파일은 `dist/` 폴더에 생성됩니다.

### 미리보기

프로덕션 빌드 미리보기:

```bash
npm run preview
```

### 린팅

코드 검증을 위한 린터 실행:

```bash
npm run lint
```

## 📁 프로젝트 구조

```
CAT_FAMILY/
├── src/
│   ├── components/        # 재사용 가능한 컴포넌트
│   │   ├── breed/         # 품종 관련 컴포넌트
│   │   ├── landing/       # 랜딩 페이지 컴포넌트
│   │   ├── layout/        # 레이아웃 컴포넌트
│   │   └── ui/            # 기본 UI 컴포넌트
│   ├── context/           # React 컨텍스트
│   ├── data/              # 데이터 및 데이터 소스
│   ├── hooks/             # 커스텀 훅
│   ├── pages/             # 애플리케이션 페이지
│   ├── utils/             # 유틸리티 및 헬퍼
│   └── breed/             # 품종 타입 및 유틸리티
├── public/                # 정적 파일
│   └── breeds/            # 품종 이미지
└── dist/                  # 컴파일된 파일
```

## 🎯 페이지

- **랜딩 페이지** (`/`): 요약 및 빠른 접근이 포함된 메인 페이지
- **품종 페이지** (`/breeds`): 필터 및 정렬이 포함된 전체 품종 목록
- **품종 상세** (`/breeds/:id`): 특정 품종의 상세 정보
- **즐겨찾기** (`/favorites`): 저장된 즐겨찾기 품종 목록
